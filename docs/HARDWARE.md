# 硬件接线方案

## 最小闭环

目标：低压直流环境下完成采集、上报、控制。

清单：

- ESP32-S3 或 ESP32-C3 开发板
- INA226 或 INA219 电压电流传感器
- 1 路继电器模块，优先选低电平触发且带光耦隔离
- 12V LED 灯带 / 小风扇 / USB 负载
- 面包板、杜邦线、万用表

## ESP32 与传感器

I2C 接线：

| ESP32 | INA226/INA219 |
|---|---|
| 3V3 | VCC |
| GND | GND |
| GPIO21 | SDA |
| GPIO22 | SCL |

负载测量：

```text
12V+ -> INA Vin+ -> INA Vin- -> 负载+ -> 负载- -> 12V-
```

## ESP32 与继电器

| ESP32 | Relay |
|---|---|
| 5V 或 3V3 | VCC |
| GND | GND |
| GPIO5 | IN |

继电器控制低压负载电源正极：

```text
12V+ -> Relay COM
Relay NO -> 负载+
负载- -> 12V-
```

## 安全边界

- 第一阶段只做 5V/12V/24V 低压直流。
- 不接 220V 市电。
- 电池必须带 BMS，充电模块必须限压限流。
- 继电器只是演示控制，真实储能充放电建议换 MOSFET/专用电源管理模块。
