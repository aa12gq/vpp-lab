# 硬件接线方案

## 当前软件状态

项目的软件侧已经可以用于硬件接入前的本地实验：

- 平台服务、simulator、Grafana、Prometheus 可跑通。
- 多设备聚合、调度预览、确认下发可用。
- 命令审计、操作审计、告警、Dashboard 可用。
- MQTT 用户名/密码、TLS、设备 HMAC 基础版可用。
- edge-gateway 支持离线缓存、本地命令下发、command/ack 捕获和边缘指标。

因此后续可以开始购买低压硬件配件。建议先买第一批，不要一次性买太多；先跑通一个真实 ESP32 节点，再扩展光伏、储能和边缘网关。

## 采购清单

### 第一批：低压直流最小套件

| 类别 | 建议组件 | 数量 | 用途 |
|---|---|---:|---|
| 主控 | ESP32-S3 DevKitC-1 或 ESP32-C3 开发板 | 2-3 | 真实设备节点，跑 MQTT、采集、控制 |
| 电流/电压采集 | INA226 I2C 电流/功率模块 | 3-5 | 采集电压、电流、功率 |
| 温度 | 防水 DS18B20 | 2 | 电池包/环境温度模拟 |
| 负载控制 | 低压 DC 继电器模块或 MOSFET 开关模块 | 2-4 | 控制 LED、风扇等可控负载 |
| 低压电源 | 12V 3A/5A 电源适配器 | 1 | 给负载和 DC 模块供电 |
| DC-DC | 12V 转 5V 降压模块 | 2 | 给开发板或外设供电 |
| 模拟负载 | 12V LED 灯带、小风扇、水泥电阻 | 若干 | 模拟家庭负载 |
| 接线 | 面包板、杜邦线、端子排、鳄鱼夹线 | 若干 | 快速搭实验电路 |
| 工具 | 万用表、USB 数据线 | 1 套 | 测量电压、电流、连线排查 |

第一批目标：让一个 ESP32 节点完成 `采集 -> MQTT -> 平台 -> Grafana/API -> 控制 -> command/ack`。

对应固件模板见 `firmware/esp32-node/`。首次烧录建议先保持平台 `DEVICE_KEYS` 为空，因为当前 ESP32 模板默认不生成 HMAC `auth` 字段；先跑通低压控制闭环，再单独升级设备签名。

### 第二批：能源与边缘扩展

| 类别 | 建议组件 | 数量 | 用途 |
|---|---|---:|---|
| 光伏模拟 | 6V/12V 小太阳能板 | 1-2 | 模拟分布式发电 |
| 光照 | BH1750 光照传感器 | 1-2 | 给光伏预测/发电模拟提供输入 |
| 储能模拟 | 18650 电池盒 + BMS + 充电模块 | 1 套 | 模拟电池储能 |
| 安全储能替代 | USB 充电宝 | 1 | 初期可替代裸电池，风险更低 |
| 边缘网关 | 树莓派 4B / 旧小主机 / N100 小主机 | 1 | 跑 edge-gateway、Mosquitto、SQLite 缓存 |
| 工业协议扩展 | USB-RS485、MAX485/TTL-RS485 模块 | 1-2 | 后续接 Modbus 设备 |

第二批目标：扩展到 PV、Battery、Load 三类资源，并把 edge-gateway 放到独立设备上运行。

## 购买顺序

1. 先买 ESP32、INA226、继电器/开关模块、12V 电源、LED/风扇、万用表。
2. 跑通一个真实负载节点：采集功率、上报 MQTT、平台展示、平台下发命令、设备回 ack。
3. 复制成 2-3 个节点：一个 PV 模拟、一个 Battery 模拟、一个 Load。
4. 再买光伏板、BH1750、储能相关模块。
5. 最后再上树莓派/小主机做边缘网关。

## 最小闭环

目标：低压直流环境下完成采集、上报、控制。

清单：

- ESP32-S3 或 ESP32-C3 开发板
- INA226 或 INA219 电压电流传感器
- 1 路继电器模块，优先选低电平触发且带光耦隔离
- 12V LED 灯带 / 小风扇 / USB 负载
- 面包板、杜邦线、万用表

## ESP32 与传感器

I2C 接线：

| ESP32 | INA226/INA219 |
|---|---|
| 3V3 | VCC |
| GND | GND |
| GPIO21 | SDA |
| GPIO22 | SCL |

负载测量：

```text
12V+ -> INA Vin+ -> INA Vin- -> 负载+ -> 负载- -> 12V-
```

## ESP32 与继电器

| ESP32 | Relay |
|---|---|
| 5V 或 3V3 | VCC |
| GND | GND |
| GPIO5 | IN |

继电器控制低压负载电源正极：

```text
12V+ -> Relay COM
Relay NO -> 负载+
负载- -> 12V-
```

## 安全边界

- 第一阶段只做 5V/12V/24V 低压直流。
- 不接 220V 市电。
- 电池必须带 BMS，充电模块必须限压限流。
- 继电器只是演示控制，真实储能充放电建议换 MOSFET/专用电源管理模块。
